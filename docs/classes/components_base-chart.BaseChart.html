    Class BaseChart

    Base chart component that provides common functionality for all chart types. This class serves as the foundation for all chart components in the library, providing basic SVG rendering, color management, and layout capabilities.

    Example

    <base-chart
      width="600"
      height="400"
      title="My Chart"
      data="[{'category': 'A', 'value': 10}, {'category': 'B', 'value': 20}]"
      showLegend="true"
      animationEnabled="true"
    ></base-chart>
