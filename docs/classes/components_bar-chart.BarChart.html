    Class BarChart

    Bar chart component that extends the base chart functionality. Displays data as vertical or horizontal bars, with support for multiple datasets, animations, and interactive features.

    Example

    <bar-chart
      width="600"
      height="400"
      title="Sales by Category"
      data="[{'category': 'Electronics', 'value': 120}, {'category': 'Clothing', 'value': 80}]"
      xKey="category"
      yKey="value"
      showLegend="true"
      showValues="true"
      animationEnabled="true"
    ></bar-chart>
